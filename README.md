# 🍽️ Zomato Restaurant Analytics Dashboard | Power BI

An interactive **Power BI Dashboard** built to analyze Zomato restaurant data and uncover valuable insights about restaurants, customer ratings, cuisines, pricing, and locations. The dashboard enables users to explore restaurant performance through dynamic visualizations, KPIs, and interactive filters for better business decision-making.

## 🚀 Features

- 📊 Executive Dashboard
- 🍴 Restaurant Performance Analysis
- 🌍 City-wise Restaurant Analysis
- ⭐ Customer Rating Analysis
- 💰 Price Range Analysis
- 🍕 Cuisine-wise Performance
- 🎯 Interactive Filters & Navigation
- 📈 Dynamic KPIs
- 🎨 Modern Microsoft/Fortune 500 Inspired UI

## 📊 Key KPIs

- Total Restaurants
- Total Cities
- Average Rating
- Average Cost for Two
- Total Cuisines
- Online Delivery Availability
- Table Booking Availability

## 📈 Dashboard Insights

- Top Cities by Number of Restaurants
- Restaurant Distribution by Rating
- Cuisine Popularity Analysis
- Average Cost by City
- Online Delivery vs No Delivery
- Table Booking Analysis
- Price Range Distribution

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Data Modeling

## 🎯 Objective

The objective of this project is to analyze restaurant data, identify customer preferences, compare restaurant performance across different cities, and provide actionable insights through an interactive and visually appealing Power BI dashboard.
